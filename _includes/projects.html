Projets

The Cube - Un plugin Minecraft pour un serveur Spigot. Ce plugin permet de jouer un mini jeu dans un labyrinthe avec un des joueurs sélectionnés aléatoirement comme étant le "Monstre" et a pour objectif d'empécher les autres de sortir.
SensorsDocker - Un conteneur développé pour fonctionner avec Node-Exporter, un module de Prometheus. Il permet de relever les températures du processeur et de les envoyer dans un fichier qui peut être lu par Node-Exporter.
Redis-Exporter - Un module de prometheus capable de se connecter à une base de donnée Redis et de chercher les informations qui y sont contenus puis de les exposés pour Prometheus.
Work In Progress
AnsibleMonitoring - Un script Ansible qui permet de déployer un système de supervision complet basé sur Prometheus et Grafana, le tout en conteneur Docker. Le but est de laisser le moins de manipulation possible à l'intégrateur.
